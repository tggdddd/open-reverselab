# Cheat Engine

开源内存扫描/修改工具。

## 用途

- 运行时内存值搜索（精确值、范围、变化趋势）
- 指针链扫描与基址定位
- 代码注入
- 速度 hack / 调试辅助

## 下载

- 官网：https://www.cheatengine.org/
- GitHub：https://github.com/cheat-engine/cheat-engine

## 安装

手动下载安装包，安装到 `tools/windows/cheat-engine/`。

## 在知识库中的引用

PE 知识库 `kb/pe-reverse/techniques/README.md` 工具映射表提到 Cheat Engine 作为内存扫描/修改的辅助工具。
