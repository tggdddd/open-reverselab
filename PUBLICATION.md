# Public migration boundary

本仓库是可公开复用版本，不是私人工作区镜像。

## 可迁移

- 通用源码、模板、合成 fixtures、无凭据配置示例；
- 公开上游项目的安装说明和来源链接；
- 已去标识化、无法反推出真实目标的技术知识。

## 禁止迁移

- 私人 `cases/`、样本、日志、截图、工具输出和分析报告；
- 真实目标标识、账号、Cookie、token、密钥、个人联系方式；
- 用户目录和机器专属绝对路径；
- 未确认再分发许可的二进制、字典、漏洞数据集和第三方源码快照。

## 发布流程

候选文件必须经过：白名单选择 → 敏感信息扫描 → diff 人工复核 → offline smoke →
fresh clone 验证。清理不是简单删除目录；README 宣称的能力必须有可执行实现或明确标为
optional。
